# Minecraft Gap Plan

最終更新: 2026-06-26

このゲームをMinecraftらしく育てるための差分リスト。各フェーズは「壊しにくく、遊びの効果が大きい順」で進める。

## Phase 1: 探索の基礎

- [done] 鉱石: 石炭、鉄、金、ダイヤを地下に生成する。
- [done] たいまつ: 建築用ブロックとして置けるようにし、近くを明るくする。
- [partial] 洞窟の探索価値: 鉱石に加えて、洞窟入口/内部の発光結晶、鍾乳石、発光結晶の欠片ドロップを追加。さらに地下遺跡ダンジョン（石レンガ部屋＋ランタン＋宝箱＋地上への階段入口）を追加し、宝箱から探索報酬がドロップする。今後は地下湖、深層エリア、地下環境音を増やす。
- [partial] 水辺の改善: 小さな池と浅い水路に加え、連続して流れる川・深い峡谷・峡谷の壁を落ちる滝を生成。今後は魚、水草、岸辺の演出を増やす。

## 地形・バイオーム（マップ拡張）

- [done] 新バイオーム: 沼地 / ジャングル / 火山 を追加。ジャングルは密で高い木、沼地は多湿で水辺が多い、火山は高所に発光する溶岩クレーター。
- [done] 自然ディテール: 岩、倒木、砂漠の枯れ木とサボテン、沼の水草をまばらに散布。
- [done] 川の連続性・峡谷・滝: うねる帯ノイズで連続して流れる川と、深い峡谷（壁に鉱石の露出、底に川/溶岩、緩やかに下る滑らかな水面）を追加。峡谷の壁ではまばらに滝（水/溶岩）が落ちる。`F10` で近くの峡谷へ移動できる。

## Phase 5: 構造物と村

- [done] 村風の小屋、畑、井戸、街灯＋役割建物: 平地/森林の平坦地に村クラスタ（井戸＋家＋畑＋道＋街灯）を生成し、各村に鍛冶屋（かまど＋溶岩火床＋金床）と市場（品物カウンター）を1軒ずつ配置。`F7` で移動。今後は村人の生活行動/取引を増やす。
- [partial] 旅人/村人風NPC: 構造物周辺や草原に村人風NPCを軽量スポーン。近くに村があれば村の家/中心付近に定住させ、村をうろつく。会話・取引・役割行動は未実装。
- [partial] チェストつき遺跡: 地下遺跡ダンジョンに宝箱を配置し、採掘で探索報酬がドロップ。今後は地上の廃墟や村にも宝箱を広げ、「開ける」インタラクション化する。
- 道路と建物のつながり改善。

## Phase 6: 地下と洞窟の強化

- 洞窟入口の見つけやすさ。
- [done] 鍾乳石風ブロック、地下湖、深層エリア: 地下湖（水/溶岩の大空洞＋発光結晶＋地上階段）と深層の良い鉱石（y≤11でダイヤ/金が出やすい）を追加。`F9` で近くの地下湖へ移動できる。
- [done] 地下遺跡ダンジョン: 石レンガ/苔石レンガの部屋、四隅の柱とランタン、中央の宝箱、地上への登り階段＋壊れた遺跡入口マーカー。`F6` で近くの地下遺跡へ移動できる。
- [done] 廃坑: 木の支柱トンネル＋たいまつ＋露出鉱石＋溶岩ハザード＋終端の宝箱＋地上入口。`F8` で近くの廃坑へ移動できる。
- [partial] 溶岩: 発光する液体ブロックとして地下に出現し、触れるとダメージ。今後は溶岩流や黒曜石化を検討。
- 鉱脈のまとまり。
- 地下の霧、暗さ、反響音。
- [done] 山の垂直化対策: 高地/雪原の高さ差を抑え、通常移動で1ブロック段差を上がれるステップアップを追加。

## 実装ルール

- 一度に巨大な仕組みを入れず、フェーズ内でも小さく検証する。
- 既存のVite/Three.js分割構成に合わせ、`src/game/parts/*.js` に追加する。
- 追加後は `npm.cmd run check` を通す。
- `HANDOFF.md` に実装済み内容と次の候補を残す。
