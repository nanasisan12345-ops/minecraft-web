# Suno 生成プロンプト（会場別・各3曲）

使い方: Suno の Custom Mode →「Style of Music」に下の1行を貼る → **Instrumental を ON** → 生成 → DL。
ファイル名は見出しの太字に合わせて `music/` に保存（対応は [README.md](README.md)）。
アーティスト名は使わない（Sunoが弾く＋著作権セーフ）。BPMはヒント程度。3曲が似たら mood/key の語を足して再生成。
