# ⛏️ 3D マインクラフト風

HTML + CSS + JavaScript（[Three.js](https://threejs.org/) / WebGL）だけで作った、ブラウザで動く3Dボクセルゲームです。ビルド不要、`index.html` を開くだけで動きます。

▶ **今すぐ遊ぶ:** https://nanasisan12345-ops.github.io/minecraft-web/

## 操作方法

| 操作 | キー |
|---|---|
| 移動 | `W` / `A` / `S` / `D` |
| ダッシュ | `Shift` |
| ジャンプ | `Space` |
| 視点 | マウス |
| ブロックを壊す | 左クリック |
| ブロックを置く | 右クリック |
| ブロック選択 | `1`〜`9` / マウスホイール |
| 一時停止 | `Esc` |

最初に画面をクリックすると始まります。

## 特徴

- **無限ワールド** — Perlinノイズによる自然な地形（山・谷・浜辺）が、プレイヤーの移動に追従して無限に生成。高さ・傾斜に応じたバイオーム（草・石・雪・砂）
- 太陽光によるリアルタイムの影（シャドウマップ）、空のグラデーション、大気の霧
- 面ごとに描き分けた32pxテクスチャ＋法線マップで、光に反応する表面の凹凸
- 流れる水面、背の高い草・花、ふっくらした木などの植生
- ブロックの破壊・設置、破壊パーティクル、効果音、ダッシュ、一時停止
- 外部依存は CDN から読み込む Three.js のみ

## 技術メモ

- 描画: Three.js (WebGL)、ブロックは種類ごとに `InstancedMesh` でまとめて高速描画
- 無限ワールド: プレイヤー周囲だけを生成するスライディングウィンドウ方式（地形は座標から決定論的に生成、編集は永続化）
- 最適化: 露出した面だけを描画する面カリング（内部の見えないブロックは描かない）
- 地形: 多重オクターブの Perlin ノイズ（fBm）
- マテリアル: `MeshPhong` ＋ Canvas API で手続き生成した32pxテクスチャ＆法線マップ
- 当たり判定 / ブロック選択: Amanatides & Woo のボクセルレイキャスト

## ローカルで動かす

`index.html` をブラウザで直接開くだけで遊べます（ブラウザのタブで開くとポインターロックが効いて快適です）。
