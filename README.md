# ⛏️ 3D マインクラフト風

HTML + CSS + JavaScript（[Three.js](https://threejs.org/) / WebGL）だけで作った、ブラウザで動く3Dボクセルゲームです。ビルド不要、`index.html` を開くだけで動きます。

<!-- PLAY_URL -->

## 操作方法

| 操作 | キー |
|---|---|
| 移動 | `W` / `A` / `S` / `D` |
| ダッシュ | `Shift` |
| ジャンプ | `Space` |
| 視点 | マウス |
| ブロックを壊す | 左クリック |
| ブロックを置く | 右クリック |
| ブロック選択 | `1`〜`9` / マウスホイール |
| 一時停止 | `Esc` |

最初に画面をクリックすると始まります。

## 特徴

- 自動生成される地形（草・土・石・砂・水・木）
- ブロックの破壊・設置
- 面ごとに描き分けたドット絵テクスチャ＋Minecraft風のフラットシェーディング
- 上空を流れる雲、破壊パーティクル、効果音、ダッシュ
- 外部依存は CDN から読み込む Three.js のみ

## 技術メモ

- 描画: Three.js (WebGL)、ブロックは種類ごとに `InstancedMesh` でまとめて高速描画
- 当たり判定 / ブロック選択: Amanatides & Woo のボクセルレイキャスト
- テクスチャ: Canvas API で 16×16 のドット絵を手続き生成

## ローカルで動かす

`index.html` をブラウザで直接開くだけで遊べます（ブラウザのタブで開くとポインターロックが効いて快適です）。
